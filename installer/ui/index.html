Bienvenido a KrakBot

KrakBot conecta tu Telegram con agentes de IA (Claude, Gemini, Codex) para que podás hacer consultas desde tu teléfono con un mensaje.

Este wizard te guía en 8 pasos para dejar el bot listo y funcionando.

⚠️ Modo reconfiguración — Ya tenés un .env existente. Los valores actuales van a precargarse en el formulario.

Dependencias npm

Instala los paquetes necesarios para que el bot funcione.

Token de Telegram

Necesitás crear un bot en Telegram para obtener el token.

  1. Abrí Telegram y buscá @BotFather
  2. Enviá el comando /newbot
  3. Seguí las instrucciones: elegí un nombre y un username
  4. Copiá el token que te da (formato: 123456:ABC-DEF...)

Agentes de IA

Detectando qué CLIs tenés instalados. Necesitás al menos uno para usar el bot.

🤖 Claude

Claude Code de Anthropic — excelente para código y análisis

⏳ Verificando...

Abrí una terminal y ejecutá: claude auth

✨ Gemini

Gemini CLI de Google — ideal para búsquedas y síntesis

⏳ Verificando...

Al ejecutar Gemini por primera vez, te va a pedir iniciar sesión con Google.

⚡ Codex

Codex CLI de OpenAI — especializado en código

⏳ Verificando...

Requiere OPENAI_API_KEY. Ver instrucciones abajo.

Configuración avanzada para Codex (OPENAI_API_KEY)

Codex CLI requiere una API key de OpenAI. Para configurarla:

  1. Obtené tu API key en platform.openai.com/api-keys
  2. Después de que el wizard escriba el .env, abrilo y agregá:
    OPENAI_API_KEY=sk-tu-clave-aqui

⚠️ Necesitás al menos un agente instalado y autenticado para continuar.

Dependencias de audio

KrakBot puede transcribir mensajes de voz usando Whisper nativo de Apple Silicon.

  • ⏳ ffmpeg verificando...
  • ⏳ mlx-whisper verificando...
⚠️ mlx-whisper requiere Apple Silicon (M1/M2/M3/M4).
En esta plataforma el audio no está disponible. Podés continuar igual — el resto del bot funciona perfectamente.
